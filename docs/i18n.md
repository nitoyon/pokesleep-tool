# Internationalization

- 5 language is supported: en, ja, ko, zh-CN, zh-TW
- `src/i18n.ts` - i18next configuration
- `src/i18n/*.json` - Translation files for 5 languages
  - `common`: Common words used by multiple applications
  - `data`: Pokemon related words
  - `events`: Event names
  - `IvCalc`: Words related to IV Calc app
  - `IvCalcNews`: IV Calc news text
  - `pokemons`: Pokemon names
  - `ResearchCalc`: Words related to Research Calc app
  - `skills`: Pokemon skill words
- Language detection in `index.tsx` based on browser language
- Dynamic font loading per language
- Translation files are split per module under `src/i18n/` subdirectories for each language
